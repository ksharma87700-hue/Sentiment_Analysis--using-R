# Sentiment_Analysis--using-R
Sentiment Analysis of TripAdvisor Reviews using R

<br>

This project performs **Sentiment Analysis** on TripAdvisor customer reviews using **R** to uncover customer opinions, emotions, and overall satisfaction. It applies Natural Language Processing (NLP) techniques to preprocess textual data, including text cleaning, tokenization, stemming, punctuation removal, stop-word elimination, and TF-IDF weighting.

<br>

The project evaluates sentiment using multiple lexicon-based methods, including **Syuzhet**, **Bing**, **AFINN**, and **NRC Emotion Lexicon**, providing comprehensive insights into positive, negative, neutral sentiments and eight core emotions. Additionally, it generates visualizations such as **word clouds**, **sentiment distribution histograms**, **emotion bar charts**, **most frequent word analysis**, and **pie charts** to effectively communicate findings.

<br>

Built using **R**, **tm**, **SnowballC**, **syuzhet**, **tidyverse**, **ggplot2**, and **wordcloud**, this project demonstrates practical skills in text mining, sentiment analysis, NLP preprocessing, data visualization, and customer feedback analytics. It serves as a valuable tool for understanding user experiences and supporting data-driven business decisions.
